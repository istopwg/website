Current Activities

The Semantic Model Working Group is responsible for the modeling of the services - Print, Copy, Scan, Fax, Email, Resource, System Control, and Transform hosted on Multifunction Devices . Standardization of the Multifunction Device (MFD) model and semantics will support interoperability of devices and services in local and enterprise networks enabling improved job submission, job management, remote administration, and support. The goal of the project is to define a unified semantic model and set of abstract operations for the most common and essential service and device features of the Multifunction Device.

Working Group History

The PWG used to have two Working Groups relating to the PWG Semantic Model.  The first group was called the Semantic Model Working Group.  That Working Group focused on the semantics of the Print Service.  It provided a single specification that summarized all the IPP specifications.  It also created an initioal XML schema limited to Print and based on IPP.  Upon publishing the PWG Semantic Model v1 the Semantic Model Working Group went into hiatus and the MFD Working Group was formed. 

The MFD Working Group extended the Semantic Models work from Print to MFD functionality.  The MFD also incorporated the Printer and related MIBs to provide access to the subunits of an MFD.  Since the MFD working group is defining Version 2 of the PWG Semantic Model the two working groups have been unified.  The currently active Working Group, MFD, has been renamed to the Semantic Model Working Group.  The content of the former Semantic Model Working Group is now available on this page.  Note that the mail list mfd@pwg.org, homepage http://www.pwg.org/mfd/ and some documents will continue use the MFD name.  In short the PWG's Semantic Model has two versions.  Version 1 is limited to the Print Service and Version 2 addresses MFDs.
