3D Printing

The Printer Working Group is investigating standardization of 3D Printing using the Internet Printing Protocol and PWG Semantic Model for the underlying network protocol and Job Ticket formats, and one or more existing high-level 3D file formats to describe the objects to print.

Summary of Work

All of our work to date has been conducted through "Birds of a Feather" (BOF) sessions at our face-to-face meetings and on the 3d-printing mailing list we host. A white paper has also been produced outlining our approach which may lead to the formation of a new workgroup or changes to an existing workgroup charter. While IPP and the PWG Semantic Model can be easily adapted to 3D printing, adapting the existing 3D file formats is proving to be more of a challenge.

3D File Formats

3D Manufacturing File Format (3MF): 3MF offers a slightly more compact XML format than AMF with physical dimensions, named materials, and shared vertices. More investigation/prototyping it needed to determine whether it is suitable for use in low-end printer controllers.
More Info

Additive Manufacturing File Format (AMF): AMF is an ISO standard XML format that supports physical dimensions, named materials, and shared vertices. It is generally considered to be the replacement for STL and is supported by some 3D software. The lack of a freely-available version of the standard has limited its use in 3D printing.
More Info

Collada (DAE): Collada is an XML format for 3D assets that is widely used in gaming. While it does offer shared vertices, it does not provide a way to name materials or define physical units.
More Info

Stereo Lithography File Format (STL): STL is the current defacto-standard file format with both plain text and binary encodings. While it is the most widely used and supported file format for 3D printing, it lacks support for physical dimensions, named materials, or shared vertices.
More Info

PDF/2.0 and Universal 3D (U3D): PDF/2.0 is expected to include 3D support using the Universal 3D format (ECMA-363). U3D is a binary file format with named materials.
