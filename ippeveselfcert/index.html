About the IPP Everywhere™ Printer Self-Certification Tools

The IPP Everywhere™ Printer self-certification tools are used to test the conformance of printers to PWG Candidate Standard 5100.14-2013: IPP Everywhere™. The testing and submission procedures are defined in PWG Candidate Standard 5100.20-2016: IPP Everywhere™ v1.0 Printer Self-Certification Manual.

Note: Tests are intended to be run on an isolated network, or at least when no other users are printing using the target printer. Otherwise the test scripts will fail randomly.

Support

The ippeveselfcert@pwg.org mailing list is used to discuss IPP Everywhere™ Printer Self-Certification. You can subscribe and view the list archives here. Issues found in the tools should be reported using the Github issues page.

PWG Raster Files

The PWG has generated PWG Raster document files for the self-certification tools in several resolutions:

  • 150dpi
  • 180dpi
  • 200dpi
  • 300dpi
  • 360dpi
  • 600dpi
  • 720dpi

Please file an issue on the Github project page if you require raster files at a different resolution.

Running the Self-Certification Tools on Windows® 7 and Higher

First download and install the current self-certification tools for Windows along with Bonjour for Windows from Apple:

  • IPP Everywhere™ Printer Self-Certification Tools v1.0 Update 4
  • IPP Everywhere™ Printer Self-Certification Tools v1.1 Release Candidate 1
  • Bonjour for Windows (Required)

Extract the PWG Raster files to your Desktop directory, then open a command prompt or power shell and run the following commands with "Printer Name" replaced by the DNS-SD service name of your printer:

cd "C:\Program Files\IPP Everywhere Printer Self-Certification Tools"
.\bonjour-tests.bat "Printer Name"
.\ipp-tests.bat "Printer Name"
.\document-tests.bat "Printer Name"

Assuming all of the tests passed, you can then run the following command to generate a JSON file containing your submission:

.\ippevesubmit "Printer Name"

Proceed to the Submitting Results instructions to submit the JSON file.

Submitting Your Self-Certification Results

If you haven't done so already, subscribe to the ippeveselfcert@pwg.org mailing list. Then send an email to the mailing list with the subject "IPP Everywhere Printer Self-Certification Results" and attach the "Printer Name.json" file created by the ippevesubmit tool to the message.

Alternately you may file an issue on Github and attach the "Printer Name.json" file to the newly created issue.
